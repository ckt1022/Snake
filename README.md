# SFML練習 貪食蛇  
## 遊戲內容  
用 上下左右 操作蛇去吃綠色食物，每吃到一個蛇身會增長，過程中不能觸碰到藍色邊界與蛇身。  
每吃到一個食物加一分，每五分level加一，並更換地圖。
